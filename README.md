# INGWEB - MarketPlus

Este repositorio contiene el proyecto **MarketPlus** dentro de la carpeta:

- `MarketPlus/`

## Documentacion principal

Revisa el README completo del proyecto aqui:

- [`MarketPlus/README.md`](./MarketPlus/README.md)

## Ejecucion rapida

### Backend
```bash
cd MarketPlus/backend
npm install
npm run dev
```

### Frontend
```bash
cd MarketPlus/frontend
npm install
npm start
```

Frontend: `http://localhost:2626`  
Backend: `http://localhost:3001`
